# Sanskrit RAG System

This project implements a CPU-based Retrieval-Augmented Generation (RAG)
system for answering questions from Sanskrit documents.

## Features
- Sanskrit document ingestion
- FAISS-based semantic retrieval
- CPU-only language model (FLAN-T5)
- Streamlit web interface

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
